# alyssais/utils

A [Homebrew](https://brew.sh) tap for little command line utilities I make.

## Installation

```sh
brew tap alyssais/utils
```

## Usage

```sh
brew install [formula]
```

See the [Formulae](Formulae) directory for a list of formulae.
